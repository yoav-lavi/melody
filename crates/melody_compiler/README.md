The Melody language compiler
</p>

## Install

```toml
[dependencies]
melody_compiler = "0.19.0"
```

## Usage

```rust
use melody_compiler::compiler;

let source = r#"1 to 5 of "A";"#;
let output = compiler(source);

assert_eq!(output.unwrap(), "A{1,5}");
```
